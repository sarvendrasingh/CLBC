Getting Started with C Programming for Complete Beginners

Hey there, if you're looking at this page, then you probably are looking to get started in C programming and quite possibly are a beginner at programming. I thank you for visiting this page, I hope that this page will be able to help you get started towards a better future in C programming. All the best! Let's get started!

List of things, we are going to need:

  1. A computer, with an OS (operating system) installed (Windows 7/8/10, Ubuntu 12.04/16.04/18.04, any other OS is also welcome :)
  2. A notepad program. (Windows-Notepad, Linux-Gedit)
  3. A Terminal ( CLI (Command Line Interface) ) program (Windows - Comamnd Prompt [ Win + R -> cmd ], Linux - Terminal [ Ctrl + Alt + T ]
  4. A compiler program.
  5. That's all! :)

If you have all of the items from the list above, congratulations, you are now ready to get started. Don't worry, if you do not have any of the resource available right now, you only need all of the above when you want to code. If you simply do not have a computer to try programming, you can ask you class advisor to allow you to use the labs for programming practice in your free time.

Installing important software

Don't worry, if you already have an Operating System installed on your computer, chances are, you already have a notepad program installed on your computer, also, the OS might also have a terminal program to interface with the computer on a hardware level.

Checking for notepad program

On a Windows machine

  1. Press 'Windows' button + R
  2. A small box will appear
  3. Click inside the box and type 'notepad' without the single quotes
  4. Press enter
  5. A white screen will open
  6. This is your notepad

On a Linux machine

  1. Press 'Windows' button
  2. A screen with a text box will appear
  3. Click inside the box and type 'gedit' or 'text' without the single quotes
  4. Press enter
  5. A white screen will open
  6. This is your notepad

Checking for terminal program

On a Windows machine

  1. Press 'Windows' button + R
  2. A small box will appear
  3. Click inside the box and type 'cmd' without the single quotes
  4. Press enter
  5. A black screen will open
  6. This is your terminal

On a Linux machine

  1. Method 1:Press 'Windows' button
  2. A screen with a text box will appear
  3. Click inside the box and type 'terminal' without the single quotes
  4. Press enter
  5. A purple screen will open
  6. This is your terminal
  7. Method 2: Press Ctrl + Alt + T

If you are able to see both the notepad program and the terminal program, then congratulations, you are now one step closer to starting to write code.All of this might seem like a big burden currently, but don't worry, this is all a one-time task. In the future, we will install an IDE that will do everything for us an make our lives easier.

Getting started with the compiler

On a Windows machine

The Windows, unfortunately, does not have a GCC package similar to the LINUX one that is shipped as-is. Instead, it has third-party versions like the TDM GCC or Cygwin GCC. But there is one problem, they are hard to configure. :( if you are new to them. But don't worry, I'll tell you the easy way to do so.

  1. Download TDM GCC from here
  2. When the file downloads, just double click on the file icon.
  3. When the software starts to install it will give you lots of windows, just press Yes or Agree or Next on every window. DO NOT change anything here.
  4. When finished the setup windows will close or ask you to close it.

You can now check whether the TDM GCC is working on your system by typing 'gcc' on the cmd terminal.

On a Linux machine

If you are on a Linux based OS, you can easily install any software from the terminal itself! So is the case with GCC. Chances are, your OS already has a version of the GCC compiler install. You can easily check it by typing 'gcc' in a terminal window. If the terminal says that gcc is not installed, you can follow the steps below.

  1. Open terminal, Ctrl + Alt + T
  2. A terminal window will open up
  3. Click inside the terminal and type
    4.                                                                         
student@student:~$ sudo apt install gcc
[sudo] password for student:

    The terminal will stop and ask for the password, just enter your login password

  4. Press enter
  5. If prompted for input, press Y
  6. When the process finishes, you will have the latest GCC version installed on your system

Congratulations!

All of the boring and hard labour inducing tasks are now done!

You are ready to write a program and test out your newly configured C programming machine.

Getting Started with Coding

Finally XD. We are now ready to try out programming in C language. For now, let's just start with a simple "Hello, world!" program. If this programs compiles and runs on your machine then it means that you have configured everything properly and everything is running fine.

The step to write a code are pretty much similar on an OS. Just open a text editor (notepad) program and start writing! The only difference you will ever see (if you are not using any IDE) that only the commands to compile the program and run the program are different on different OS.

On a Windows machine

Simply open a notepad and write the following code. After you are done, save you file as hello.c Please understand that the hello in the name of the file is just for demo. You can use any name that you want. Surprise a friend by naming your first C program by naming the file after them. Just remember to write the '.c' extension always.

  1. Open the notepad program of your choice, ex - Notepad.
  2. Type the following code into your Notepad. 
    3.                                                                         
#include<stdio.h>
int main()
{
    printf("Hello, world!");
    return 0;
}
  3. Now, save the file as hello.c . Please note that you can give any name for the file. Ex -
    • hello.c
    • file.c
    • batman.c
    • SEE? ANY NAME YOU WANT!
  4. Your file is now probably saved. Go to the location you saved your file and check that it is actually saved as the filename.c
  5. Now we just have to compile the program you just wrote in C. We call this process 'Compilation'. It is necessary to convert the C Code to the Equivalent Binary code that the computer can understand.
  6. The command for compiling the C code in Windows Command Prompt via GCC is 
                                                                                
C: gcc hello.c
    This command will compile the C code and convert the C program to equivalent binary code. If you like you can execute the command as 'gcc hello.c -o hello.exe'. Please note, the binary code will be saved to the new file mentioned after the '-o' in the command. Like we have written -o hello.exe here, hence a file name hello.exe will be created that will contain all the binary code. We have to mention the '.exe' in Windows and not in Linux, because Windows' default file extension in .exe : Executable.
  7. The command for running the binary code in Windows Command Prompt via GCC is 
                                                                                
C: a.exe
    This command will call the file 'a.exe' present in the same folder and run the binary code present in the file. Please note that the compiler will name the output file 'a.exe' by default if you do not tell the compiler to give it another name. You can give another name for the output file by using '-o filename.exe' after the 'gcc' command ex - 
                                                                                
C: gcc hello.c -o hello.exe
    This will save the file as hello.exe and you can run the file with the name hello.exe.
  8. If you have followed all the above steps properly, then the program output should be shown in the Windows Command Prompt

On a Linux machine

Simply open a notepad and write the following code. After you are done, save you file as hello.c Please understand that the hello in the name of the file is just for demo. You can use any name that you want. Surprise a friend by naming your first C program by naming the file after them. Just remember to write the '.c' extension always.

  1. Open the notepad program of your choice, ex - Notepad.
  2. Type the following code into your Notepad. 
    3.                                                                         
#include<stdio.h>
int main()
{
    printf("Hello, world!");
    return 0;
}
  3. Now, save the file as hello.c . Please note that you can give any name for the file. Ex -
    • hello.c
    • file.c
    • batman.c
    • SEE? ANY NAME YOU WANT!
  4. Your file is now probably saved. Go to the location you saved your file and check that it is actually saved as the filename.c
  5. Now we just have to compile the program you just wrote in C. We call this process 'Compilation'. It is necessary to convert the C Code to the Equivalent Binary code that the computer can understand.
  6. The command for compiling the C code in Linux terminal via GCC is 
                                                                                
student@student:~$ gcc hello.c
    This command will compile the C code and convert the C program to equivalent binary code.You can also use '-o filename' after the 'gcc' command. Please note, the binary code will be saved to the new file mentioned after the '-o' in the command. Like we have written -o hello here, hence a file name hello will be created that will contain all the binary code. We do not have to type the '.exe' like we have to do in Windows, Linux understands that a file with no extension is a binary file.
  7. The command for running the binary code in Linux Terminal via GCC is 
                                                                                
student@student:~$ ./hello
    This command will call the file 'hello' present in the same folder and run the binary code present in the file.
  8. If you have followed all the above steps properly, then the program output should be shown in the Linux Terminal
