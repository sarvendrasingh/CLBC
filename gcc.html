The GCC Compiler

Hello learners! This page is all about the GCC Compiler. If you would like to read the official documentation on the GCC Compiler, you can view it here. If you want, you can download the compiler form this link here.

The GCC (GNU Compiler Collection) Compiler is a program that compiles the source code written in a programming language. Simply it converts instructions written in programming languages by a user into machine/binary language so that the computer can understand it. Originally it was written for compiling only C programs, but later on, support for other programming languages have since been added.

The GNU Compiler Collection includes front ends for C, C++, Objective-C, Fortran, Ada, and Go, as well as libraries for these languages (libstdc++,...). GCC was originally written as the compiler for the GNU operating system. The GNU system was developed to be 100% free software, free in the sense that it respects the user's freedom.

Using GCC on the terminal

GCC runs right on the terminal of your Operating System (Windows, Linux, Unix, Mac). If you already have GCC installed on your system, just open up your terminal and type gcc to check it. 

                                        
student@student:~$ gcc

That's it! If your terminal says something like 

                            
student@student:~$ gcc
gcc: fatal error: no input files
compilation terminated.

then you're all set to write your programs and compile them in your terminal.

Compiling A Simple Hello World C Program with GCC

On a Windows machine

  1. Just open any notepad program or IDE of your choice.
  2. Write the "Hello, World" program in C, given below
    3.                                                                         
#include<stdio.h>
int main()
{
    printf("Hello, world!");
    return 0;
}
  3. Save the file as hello.c
  4. Open the command prompt
  5. Go to the diretory where your file is located
  6. Now type 
    7.                                     
student@student:~$ gcc hello.c -o hello.exe
student@student:~$ hello.exe
  7. If the program runs on the command prompt as shows the output, then your gcc installation has been successful.

  1. Just open any notepad program or IDE of your choice.
  2. Write the "Hello, World" program in C, given below
